Bomb Defusal Manual

By Wixonic

Introduction

A bomb will explode when its countdown timer reaches 00:00 or when too many strikes have been recorded.
The only way to defuse a bomb is to disarm all of its modules before its countdown timer expires.

Each bomb includes three modules that must be disarmed.
Instructions for disarming modules can be found in the manual.

When you make a mistake, the bomb will record a strike which will be displayed on the indicator below the countdown timer.
Bomb will explode upon the second strike.

Wires

Only the one correct wire needs to be cut to disarm the module. A wire module can have three or four wires on it. A wire can be blue, red, or yellow. Wire ordering begins with the first on the top.

Three wires

If the bomb is white or red, if the first wire is yellow, and if the last wire isn't red, cut the second wire.
Otherwise, if the bomb is white or red, if the first wire is blue, and if the second wire isn't yellow, cut the first wire.
Otherwise, if the bomb is white or red, and if the second wire is red, cut the second wire.
Otherwise, if the bomb is white or red, and if the last wire is yellow, cut the last wire.
Otherwise, if the bomb is white or red, cut the first wire.

Otherwise, if the bomb is black or blue, if the first wire is red, and if the last wire isn't red, cut the last wire.
Otherwise, if the bomb is black or blue, if the first wire is blue, and if the last wire isn't yellow, cut the second wire.
Otherwise, if the bomb is black or blue, and if the second wire is yellow, cut the last wire.
Otherwise, if the bomb is black or blue, and if the second wire isn't red, cut the second wire.
Otherwise, if the bomb is black or blue, cut the first wire.

Four wires

If the bomb is white or red, if the second wire is yellow, and if the last wire isn't red, cut the second wire.
Otherwise, if the bomb is white or red, if the second wire isn't red, and if the third wire is yellow, cut the first wire.
Otherwise, if the bomb is white or red, and if the first wire is blue, cuth the last wire.
Otherwise, if the bomb is white or red, and if the second wire isn't yellow, cut the third wire.
Otherwise, if the bomb is white or red, cut the last wire.

If the bomb is black or blue, if the second wire is red, and if the third wire isn't red, cuth the first wire.
Otherwise, if the bomb is black or blue, if the first wire is yellow, and if the second wire isn't blue, cut the last wire.
Otherwise, if the bomb is black or blue, and if the last wire isn't yellow, cut the second wire.
Otherwise, if the bomb is black or blue, and if the first wire is red, cut the first wire.
Otherwise, if the bomb is black or blue, cut the third wire.
